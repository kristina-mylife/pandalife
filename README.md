# 🐼 PandaLife

Однопользовательская игра: панда ходит и прыгает по бесконечному миру, собирая пельмени!

## Игровой процесс

- **Бесконечный мир** — при движении вправо появляется новое пространство с платформами и пельменями
- **Ходьба и прыжки** — ровная земля для прогулок и платформы для прыжков
- **← →** или **A D** — движение
- **↑ W Пробел** или **тап** — прыжок
- Собирай пельмени 🥟 на земле и на платформах
- 3 жизни (падение в пропасть или в воду = минус жизнь)
